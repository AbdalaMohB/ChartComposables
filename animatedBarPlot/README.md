# Animated Bar Plot

This is a bar plot composable for displaying data.

## Input Parameters

* modifier: Only affects canvas
* data: A list of name and list of data column pairs
* dataColors: A list of colors for columns
* showNumbers: Whether or not to show numbers on the Y axis
* step: Step increase for numbers shown on Y axis
* maxRecordedValueSlider: Affects height of Y axis line and max recorded value on said line
* barWidth: Controls the width of the bars in the plot
* barMaxHeightPercentage: Controls how much of the canvas height the plot takes
* barCornerRadius: Controls the roundness of the bars
* textStyle: Controls the text style of all text
* lineColor: Controls the color of the plot lines
